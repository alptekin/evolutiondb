What's In the Box?

EvolutionDB is a single C binary. It is built to be long-term memory for AI agents. The memory primitives, native vectors, and time-travel queries live inside the engine. You reach all of it through the PostgreSQL wire protocol.

psychology

Agent-Memory Primitives

SQL primitives give agents long-term memory. Storing what an agent knows is a core part of the storage engine. It is not an extra layer or a side service.

Native Vectors & HNSW

The engine has a built-in VECTOR(N) type and an HNSW index. You can run hybrid search over meaning and words. You do not need a separate vector service.

history

Time-travel Queries

An MVCC engine keeps older versions of your data. So you can query the past with AS OF. You can ask what an agent knew at any time before now.

SELECT * FROM memories
AS OF '2026-01-01';
hub

One Binary, Dual Protocol

One server speaks two protocols. It uses the PostgreSQL wire protocol on port 5433. It also uses a native EVO text protocol on port 9967. You can connect with psql or DBeaver. You can also run the embedded mode with no Docker.

PG WIRE :5433 EVO :9967 EMBEDDED MODE
Server up · dual protocol
$ psql -h 127.0.0.1 -p 5433 -U admin -d testdb
$ evosql-cli # native EVO on :9967

A Single C Binary

One app can replace a whole stack of extra tools. The storage is unified and lives in the engine. It uses 4KB pages, slotted pages, B+ tree indexes, and a Clock-Sweep buffer pool. It also has a write-ahead log with crash recovery.

terminal ./evosql-server

Built for Production

groups

Multi-tenant Isolation

Each tenant gets its own DATABASE, USER, and GRANT. The engine checks rights step by step. It checks the table first, then the schema, then the database.

lock

Security Built In

Pages use transparent data encryption with AES-256. Passwords use PBKDF2-SHA256. Security lives in the storage layer itself.

neurology

Agent-native

It ships an MCP server and a standalone agent. Every action waits for a human. Drafts go into a queue first. A person approves them before anything is sent.
