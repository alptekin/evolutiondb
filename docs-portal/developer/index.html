DEVELOPER PORTAL

Build on EvolutionDB

Here you find everything you need to give your app or agent long-term memory. You get language SDKs, ready-made data connectors, platform packages, and the wire-protocol reference. Every package here is open source under the MIT license. They all live in the monorepo.

Quick start

Run the database first. It is one binary. It speaks the PostgreSQL wire protocol on :5433 and the native EVO protocol on :9967. Then you can talk to it from the language you like.

# 1. start EvolutionDB
docker compose up -d

# 2. store & recall memory from Python
pip install python-evosql-memory
from evosql_memory import MemoryClient
m = MemoryClient(host="127.0.0.1", port=5433)
m.save("Alice prefers email over Slack", tags=["alice"])
m.search("how does Alice like to be contacted?")
