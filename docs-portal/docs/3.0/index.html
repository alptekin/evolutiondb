EvolutionDB 3.0 Documentation

This is the official reference manual for EvolutionDB. EvolutionDB is a SQL engine. It is built to be long-term memory for AI agents. This manual covers version 3.0. Version 3.0 is the current version.

info
EvolutionDB has only one version line today. That line is 3.0. The version selector is at the top left. It will list more versions when we release them. Each version will have its own copy of this manual.
