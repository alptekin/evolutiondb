Welcome to EvolutionDB Server

EvolutionDB is an open source SQL engine designed to act as the long-term memory layer for AI assistants. This installer sets up the server so that Claude, ChatGPT, GitHub Copilot, and any other MCP-aware client can keep persistent memory across conversations on this Mac.

What this installer does

  • Copies the evosql-server binary to /usr/local/bin
  • Creates a data directory at /usr/local/share/evosql
  • Installs a system LaunchDaemon at /Library/LaunchDaemons/com.evolutiondb.server.plist
  • Starts the server on login and keeps it running across reboots

What it does not do

  • Open any network ports beyond the local loopback interface (127.0.0.1:5433 and 127.0.0.1:9967)
  • Send any data to a remote server
  • Modify your existing PostgreSQL installation, if any

You can uninstall everything later by running sudo /usr/local/share/evosql/uninstall.sh.
