EvolutionDB is running.

The server is now active in the background and will start automatically every time this Mac boots. You can connect to it from any PostgreSQL-compatible client.

Files placed on disk

PathPurpose
/usr/local/bin/evosql-server The database engine binary
/usr/local/share/evosql/ Data directory (DB file, WAL, etc.)
/usr/local/share/evosql/uninstall.sh Script that removes everything this installer created
/Library/LaunchDaemons/com.evolutiondb.server.plist System-level LaunchDaemon that keeps the server up

Connect to it

psql -h 127.0.0.1 -p 5433 -U admin -d evosql
# password: admin

Or open DBeaver / TablePlus / pgAdmin and point them at 127.0.0.1:5433 with username admin and password admin.

Wire it into Claude Desktop

Drop this block into ~/Library/Application Support/Claude/claude_desktop_config.json:

{
  "mcpServers": {
    "evolutiondb-memory": {
      "command": "uvx",
      "args": ["mcp-server-evolutiondb"],
      "env": {
        "EVOSQL_HOST": "127.0.0.1",
        "EVOSQL_PORT": "5433",
        "EVOSQL_USER": "admin",
        "EVOSQL_PASSWORD": "admin",
        "EVOSQL_DATABASE": "evosql",
        "MCP_USER_ID": "you",
        "MCP_STORE_PREFIX": "mcp"
      }
    }
  }
}

Restart Claude Desktop with Cmd Q and start a new conversation. Memory now persists across sessions.

Stop or remove the server

sudo launchctl bootout system /Library/LaunchDaemons/com.evolutiondb.server.plist  # stop
sudo /usr/local/share/evosql/uninstall.sh                                          # uninstall

Default credentials are admin/admin. Change them with ALTER USER admin PASSWORD '<new>' from psql when you're ready for a real workload.
