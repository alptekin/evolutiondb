EvolutionDB has been deployed!

Connect via PostgreSQL wire protocol (port {{ .Values.service.pgPort }}):
  # Native EVO protocol is also available on port {{ .Values.service.evoPort }}

  # From inside the cluster
  psql -h {{ include "evolutiondb.fullname" . }}.{{ .Release.Namespace }}.svc.cluster.local -p {{ .Values.service.pgPort }} -U {{ .Values.auth.username }} -d testdb

  # Port-forward to localhost
  kubectl port-forward svc/{{ include "evolutiondb.fullname" . }} {{ .Values.service.pgPort }}:{{ .Values.service.pgPort }} {{ .Values.service.evoPort }}:{{ .Values.service.evoPort }} -n {{ .Release.Namespace }}
  psql -h localhost -p {{ .Values.service.pgPort }} -U {{ .Values.auth.username }} -d testdb
